### Make It Easy
```
‚ùå "Fill out the form on my website"
‚úÖ "Comment 'guide' and I'll DM you"

‚ùå "Check out my latest blog post"
‚úÖ "Link in first comment. Takes 3 minutes to read."

‚ùå "Schedule a call if interested"
‚úÖ "DM me 'talk' and I'll send my calendar link"
```

### The "Keyword" Trick
```
Why "DM me '[keyword]'" works:

1. Lower barrier than explaining what they want
2. Easy for you to filter/identify leads
3. Creates micro-commitment
4. Fun/memorable

Examples:
- DM me "blueprint" for the free guide
- Comment "framework" for the template
- Reply "podcast" and I'll share the episode
```

### Stacking CTAs
```
Offer multiple ways to engage:

Primary CTA: [Engagement goal]
Secondary CTA: [Growth goal]
Tertiary CTA: [Lead generation goal]

Example:
"What would you add to this list? üëá
(And follow for more leadership content.
DM me 'toolkit' for my free resource.)"
```

## CTA Testing Framework

### What to Test
```
1. CTA type (question vs command)
2. Placement (end vs middle vs P.S.)
3. Keyword choice
4. Specificity level
5. Single vs multiple CTAs
```

### How to Measure
```
Track across similar posts:
- Comment volume
- DM volume
- Profile visits
- Follower growth
- Link clicks (if applicable)
```

### Iteration Process
```
1. Use same content type
2. Change only the CTA
3. Post at similar times
4. Track results
5. Adopt winner
6. Test new variation
```
