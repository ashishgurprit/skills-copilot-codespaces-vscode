# Promotional LinkedIn Post Templates

Soft-sell templates for promoting products, services, and offers without being salesy.

---

## The Golden Rule of LinkedIn Promotion

```
90% value, 10% promotion.

Never promote without first establishing value.

These templates wrap promotion in genuine value.
```

## Promotional Best Practices

### The Promotion Ratio

```
For every 1 promotional post:
‚Ä¢ 9 value-giving posts
‚Ä¢ Or wait at least 2 weeks between promotions

Your audience will forgive promotion
if you consistently deliver value.
```

### Soft vs Hard CTAs

```
Soft CTAs (preferred for LinkedIn):
‚Ä¢ "DM me [keyword] for more info"
‚Ä¢ "Comment [word] if you want the link"
‚Ä¢ "Check out the link in comments"
‚Ä¢ "Curious? Let's connect"

Hard CTAs (use sparingly):
‚Ä¢ "Buy now"
‚Ä¢ "Sign up today"
‚Ä¢ "Don't miss out"
‚Ä¢ "Limited time"
```

### Where to Put Links

```
Option 1: First comment (most common)
‚Ü≥ Add link immediately after posting
‚Ü≥ Slightly lower reach than link-free
‚Ü≥ But allows for clickable link

Option 2: In the post (lowest reach)
‚Ü≥ LinkedIn deprioritizes posts with links
‚Ü≥ Only for truly high-value destinations

Option 3: Tell them to DM for link
‚Ü≥ Highest reach
‚Ü≥ Creates personal connection
‚Ü≥ More effort to follow up

Option 4: "Link in profile"
‚Ü≥ Sends traffic to your profile
‚Ü≥ Feature section should have the link
```

### Making Promotion Feel Natural

```
‚úÖ Do:
‚Ä¢ Lead with value/story
‚Ä¢ Show genuine results
‚Ä¢ Be transparent about what you're promoting
‚Ä¢ Acknowledge it's promotional (people respect honesty)
‚Ä¢ Make the CTA optional/low pressure

‚ùå Don't:
‚Ä¢ Start with the pitch
‚Ä¢ Use fake scarcity
‚Ä¢ Over-hype
‚Ä¢ Spam the same promotion
‚Ä¢ Make it all about you/your product
```

### Testimonial Best Practices

```
‚úÖ Strong testimonials:
‚Ä¢ Specific results with numbers
‚Ä¢ Named person/company (with permission)
‚Ä¢ Addresses a specific problem
‚Ä¢ Feels authentic (not polished PR)

‚ùå Weak testimonials:
‚Ä¢ "Great product!"
‚Ä¢ Anonymous quotes
‚Ä¢ Vague benefits
‚Ä¢ Overly polished language
```

### The "Problem First" Approach

```
Structure promotions like this:

1. The Problem (that your audience has)
2. The Pain (why it matters)
3. The Solution (introduced naturally)
4. The Proof (results/testimonials)
5. The CTA (soft, optional)

Never start with your product.
Start with their problem.
```
