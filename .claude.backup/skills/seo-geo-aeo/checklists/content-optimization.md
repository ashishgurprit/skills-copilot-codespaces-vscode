# Content Optimization Checklist

Comprehensive checklist for optimizing content for SEO, GEO, and AEO.

## Pre-Writing Research

### Keyword Research
- [ ] Primary keyword identified
- [ ] Search volume verified
- [ ] Keyword difficulty assessed
- [ ] Search intent determined (I/N/C/T)
- [ ] Secondary keywords listed (3-5)
- [ ] Long-tail variations identified
- [ ] Related questions gathered (PAA)

### SERP Analysis
- [ ] Top 10 results analyzed
- [ ] Content gaps identified
- [ ] Featured snippet opportunity assessed
- [ ] SERP features noted (PAA, images, videos)
- [ ] Average word count of top results
- [ ] Content format of top results

### Competitor Content Audit
- [ ] Top 3 competitor articles reviewed
- [ ] Unique angles identified
- [ ] Missing topics found
- [ ] Quality benchmark established

---

## Content Structure

### Outline Creation
- [ ] H1 includes primary keyword
- [ ] Logical H2-H4 hierarchy planned
- [ ] All user questions addressed
- [ ] Unique value/angle defined
- [ ] Comprehensive topic coverage

### Content Framework
```
H1: [Primary Keyword] - Compelling Title
│
├── Introduction (hook + keyword in first 100 words)
│
├── H2: [Secondary Keyword Section]
│   ├── H3: Subsection
│   └── H3: Subsection
│
├── H2: [Secondary Keyword Section]
│   ├── H3: Subsection
│   └── H3: Subsection
│
├── H2: FAQ Section (from PAA)
│   ├── H3: Question 1?
│   ├── H3: Question 2?
│   └── H3: Question 3?
│
└── Conclusion (CTA + summary)
```

## On-Page SEO Elements

### Title Tag
- [ ] Primary keyword included
- [ ] Keyword front-loaded (first 60 chars)
- [ ] Length: 50-60 characters
- [ ] Compelling for CTR
- [ ] Unique (not duplicated elsewhere)
- [ ] Brand included (if space)

### Meta Description
- [ ] Primary keyword included
- [ ] Length: 150-160 characters
- [ ] Compelling value proposition
- [ ] Call-to-action included
- [ ] Unique (not duplicated)
- [ ] Accurate content preview

### URL/Slug
- [ ] Primary keyword included
- [ ] Lowercase only
- [ ] Hyphens between words
- [ ] No special characters
- [ ] Under 75 characters
- [ ] Stop words removed

### Heading Optimization
- [ ] Single H1 per page
- [ ] Primary keyword in H1
- [ ] Secondary keywords in H2s
- [ ] Logical hierarchy (no skipped levels)
- [ ] Scannable and descriptive
- [ ] Question format where relevant

---

## Content Quality

### Writing Standards
- [ ] Clear, concise writing
- [ ] Active voice preferred
- [ ] Short paragraphs (2-4 sentences)
- [ ] Varied sentence length
- [ ] No jargon (or explained if necessary)
- [ ] Appropriate reading level for audience
- [ ] Grammar and spelling checked

### E-E-A-T Signals
- [ ] **Experience**: First-hand knowledge demonstrated
- [ ] **Expertise**: Author credentials shown
- [ ] **Authoritativeness**: Cited by sources / citing sources
- [ ] **Trustworthiness**: Accurate, verifiable information

### Author Information
- [ ] Author name displayed
- [ ] Author bio included
- [ ] Author credentials mentioned
- [ ] Author photo (optional but recommended)
- [ ] Author schema markup

### Content Freshness
- [ ] Publication date visible
- [ ] "Last updated" date (if updated)
- [ ] Current year statistics
- [ ] Recent examples/references
- [ ] No outdated information

---

## Keyword Integration

### Keyword Placement
- [ ] Primary keyword in title tag
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in last paragraph
- [ ] Secondary keywords in H2/H3
- [ ] Keywords in image alt text
- [ ] Natural keyword density (1-2%)

### Semantic Keywords
- [ ] Related terms included
- [ ] Synonyms used naturally
- [ ] Topic comprehensively covered
- [ ] No keyword stuffing
- [ ] LSI keywords integrated

---

## Internal Linking

### Link Strategy
- [ ] 3-5 internal links per 1000 words
- [ ] Links to related content
- [ ] Links to pillar/cornerstone content
- [ ] Links from high-authority pages
- [ ] Descriptive anchor text (not "click here")
- [ ] Links placed contextually (not forced)

### Hub & Spoke Model
- [ ] Content linked to relevant hub/pillar
- [ ] Related spoke content interlinked
- [ ] Breadcrumb navigation present
- [ ] Clear site hierarchy maintained

---

## External Linking

### Outbound Links
- [ ] Links to authoritative sources
- [ ] Links support claims/statistics
- [ ] Links open in new tab (optional)
- [ ] No broken external links
- [ ] Nofollow for paid/sponsored links
- [ ] 2-5 external links typical

### Source Quality
- [ ] Government sites (.gov)
- [ ] Educational institutions (.edu)
- [ ] Industry authorities
- [ ] Peer-reviewed research
- [ ] Reputable news sources
- [ ] Official statistics

---

## Multimedia Optimization

### Images
- [ ] Relevant images included
- [ ] Original/unique images preferred
- [ ] Descriptive file names (keyword.jpg)
- [ ] Alt text with keywords (natural)
- [ ] Compressed for web (< 200KB)
- [ ] WebP format preferred
- [ ] Correct dimensions (no scaling)
- [ ] Lazy loading implemented
- [ ] Captions where helpful

### Video
- [ ] Video embedded if relevant
- [ ] Video transcript provided
- [ ] Video schema markup
- [ ] Thumbnail optimized
- [ ] Video supports content (not filler)

### Infographics/Charts
- [ ] Data visualizations where helpful
- [ ] Shareable format
- [ ] Embed code provided
- [ ] Alt text for accessibility
- [ ] Source attribution

---

## GEO Optimization (AI Search)

### Quotability
- [ ] Clear, standalone statements
- [ ] Definitive facts included
- [ ] Statistics with sources
- [ ] Expert quotes attributed
- [ ] Concise definitions

### Content Structure for AI
- [ ] "What is [topic]?" section
- [ ] Numbered lists for processes
- [ ] Tables for comparisons
- [ ] Bullet points for features
- [ ] FAQ section included

### Source Attribution
- [ ] Academic sources cited
- [ ] Data sources linked
- [ ] Expert credentials mentioned
- [ ] Publication dates included
- [ ] Methodology disclosed (for original research)

## AEO Optimization (Featured Snippets)

### Featured Snippet Targeting
- [ ] Question identified for targeting
- [ ] Answer directly below question heading
- [ ] Answer length: 40-60 words (paragraph)
- [ ] Or numbered list (5-8 items)
- [ ] Or table format (comparison)

### Voice Search Optimization
- [ ] Conversational language used
- [ ] Question-answer format
- [ ] Local modifiers included
- [ ] "Near me" alternatives
- [ ] Natural speaking patterns

### FAQ Implementation
- [ ] FAQ section included
- [ ] Questions from PAA answered
- [ ] Concise, direct answers
- [ ] FAQ schema markup added

- [ ] Author schema

- [ ] BreadcrumbList schema

- [ ] Form labels present

## Pre-Publish Checklist

### Final Review
- [ ] Spelling and grammar checked
- [ ] All links working
- [ ] Images displaying correctly
- [ ] Schema validated (Google Rich Results Test)
- [ ] Mobile preview checked
- [ ] Meta information complete
- [ ] Social sharing preview correct

### Technical Verification
- [ ] Canonical URL set
- [ ] Index/follow status correct
- [ ] Open Graph tags present
- [ ] Twitter Card tags present
- [ ] Page loads under 3 seconds

---

## Post-Publish Tasks

### Indexing
- [ ] Submit to Google Search Console
- [ ] Submit to Bing Webmaster Tools
- [ ] Check for indexing within 48 hours
- [ ] Verify in site:search

### Promotion
- [ ] Share on social media
- [ ] Internal link from relevant pages
- [ ] Email newsletter (if applicable)
- [ ] Update sitemap

### Monitoring
- [ ] Track rankings for target keywords
- [ ] Monitor Search Console impressions
- [ ] Check for crawl errors
- [ ] Monitor page performance

---

## Content Update Checklist

### When to Update
- [ ] Rankings declining
- [ ] Information outdated
- [ ] New developments in topic
- [ ] Competitor content improved
- [ ] 6-12 months since publication

### Update Process
- [ ] Review current performance
- [ ] Identify improvement opportunities
- [ ] Update statistics/dates
- [ ] Add new sections if needed
- [ ] Improve existing content
- [ ] Add new internal links
- [ ] Update "last modified" date
- [ ] Re-submit to search engines
