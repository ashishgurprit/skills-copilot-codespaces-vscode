# Local SEO Optimization Checklist

Complete checklist for optimizing local search visibility across all major platforms.

## Google Business Profile (GBP)

### Profile Setup & Verification
- [ ] Claim or create Google Business Profile
- [ ] Complete verification (postcard, phone, email, or video)
- [ ] Set up manager access for team members
- [ ] Connect Google Ads account (if applicable)

### Essential Information
- [ ] Business name (exact legal name, no keywords stuffed)
- [ ] Primary category (most specific available)
- [ ] Secondary categories (all relevant, max 9)
- [ ] Address (exact match with website)
- [ ] Service area (if service-area business)
- [ ] Phone number (local, trackable)
- [ ] Website URL (homepage or location page)
- [ ] Business hours (accurate, including holidays)
- [ ] Special hours (holidays, events)

### Description & Attributes
- [ ] Business description (750 chars, keywords natural)
- [ ] From the business description
- [ ] All relevant attributes selected
- [ ] Service/product offerings listed
- [ ] Health & safety attributes (if applicable)
- [ ] Accessibility attributes
- [ ] Payment methods
- [ ] Amenities

### Visual Content
- [ ] Logo uploaded (square, high quality)
- [ ] Cover photo (landscape, brand representative)
- [ ] Interior photos (5-10)
- [ ] Exterior photos (3-5, different angles)
- [ ] Team photos (with permission)
- [ ] Product/service photos (10+)
- [ ] Videos (if available, under 30 seconds)
- [ ] 360 virtual tour (if applicable)

### Products & Services
- [ ] All services listed with descriptions
- [ ] Service pricing (if applicable)
- [ ] Products with photos and prices
- [ ] Product categories organized
- [ ] Links to relevant website pages

### Posts & Updates
- [ ] Welcome/intro post created
- [ ] Regular posts (weekly recommended)
- [ ] Event posts for promotions
- [ ] Offer posts for specials
- [ ] Product posts for new items
- [ ] COVID/safety updates (if relevant)

## Website Local Optimization

### NAP Consistency
- [ ] Name, Address, Phone in footer (all pages)
- [ ] Contact page with full details
- [ ] Embedded Google Map
- [ ] Click-to-call phone link
- [ ] Directions link

### Local Schema Markup
- [ ] LocalBusiness schema on homepage
- [ ] Organization schema
- [ ] Address schema (PostalAddress)
- [ ] GeoCoordinates
- [ ] OpeningHoursSpecification
- [ ] Review schema (if applicable)
- [ ] Service schema for each service

**LocalBusiness Schema Template**:
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "@id": "https://yoursite.com/#business",
  "name": "Business Name",
  "image": "https://yoursite.com/logo.png",
  "url": "https://yoursite.com",
  "telephone": "+61-X-XXXX-XXXX",
  "priceRange": "$$",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "123 Main Street",
    "addressLocality": "Sydney",
    "addressRegion": "NSW",
    "postalCode": "2000",
    "addressCountry": "AU"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": -33.8688,
    "longitude": 151.2093
  },
  "openingHoursSpecification": [
    {
      "@type": "OpeningHoursSpecification",
      "dayOfWeek": ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday"],
      "opens": "09:00",
      "closes": "17:00"
    }
  ],
  "sameAs": [
    "https://facebook.com/yourbusiness",
    "https://instagram.com/yourbusiness",
    "https://linkedin.com/company/yourbusiness"
  ]
}
```

## Review Management

### Review Generation
- [ ] Review link created (Google, Facebook, etc.)
- [ ] Review request email template created
- [ ] Post-service review request process
- [ ] QR code for in-location reviews
- [ ] Review request timing optimized

### Review Response
- [ ] All reviews responded to (within 24-48 hours)
- [ ] Positive review response template
- [ ] Negative review response template
- [ ] Neutral review response template
- [ ] Owner response includes keywords naturally

### Review Monitoring
- [ ] Google review alerts enabled
- [ ] Facebook notification enabled
- [ ] Third-party review sites monitored
- [ ] Review management tool (if needed)

### Response Templates

**Positive Review**:
```
Thank you [Name] for your wonderful feedback! We're so pleased that
[specific mention from review]. Our team at [Business Name] in [Location]
takes pride in [value proposition]. We look forward to serving you again!
```

**Negative Review**:
```
Thank you [Name] for sharing your experience. We're sorry to hear that
[acknowledge specific issue]. At [Business Name], we strive to [standard].
Please reach out to us at [email/phone] so we can address your concerns
and make things right.
```

- [ ] Wrong category selection
- [ ] Neglecting review responses
- [ ] Outdated business hours

- [ ] Thin location pages
- [ ] Ignoring negative reviews
