# Keyword Research Methodology

Complete guide to keyword research for SEO, GEO, and AEO optimization.

## Keyword Research Framework

### 1. Seed Keyword Generation

**Sources for Seed Keywords**:
- Business offerings/services
- Customer questions and feedback
- Competitor analysis
- Industry terminology
- Product features and benefits

**Brainstorming Template**:
```
Business: [Your business]
Core offering: [Main product/service]

Seed keywords:
- What we do: [list 5-10]
- What customers call it: [list 5-10]
- Problems we solve: [list 5-10]
- Industry terms: [list 5-10]
```

### 2. Keyword Expansion

**Methods**:

| Method | Tool/Approach | Output |
|--------|---------------|--------|
| Autocomplete | Google/Bing search bar | Long-tail variations |
| Related searches | Bottom of SERP | Semantic variations |
| People Also Ask | Google PAA boxes | Question keywords |
| Competitor analysis | Ahrefs/SEMrush | Competitor keywords |
| Topic clustering | Answer the Public | Question modifiers |

**Modifier Categories**:

| Type | Examples | Use Case |
|------|----------|----------|
| Informational | how to, what is, guide, tutorial | Blog content |
| Commercial | best, top, review, comparison | Buying guides |
| Transactional | buy, price, discount, near me | Product pages |
| Navigational | [brand name], login, contact | Brand pages |

### 3. Keyword Metrics

**Essential Metrics**:

| Metric | Definition | Target |
|--------|------------|--------|
| Search Volume | Monthly searches | Depends on niche |
| Keyword Difficulty | Competition level (0-100) | < 40 for new sites |
| CPC | Cost per click (paid) | Indicates commercial value |
| SERP Features | Rich results present | Opportunity indicator |
| Trend | Search trend direction | Upward preferred |

**Volume Guidelines by Niche**:

| Niche Type | Good Volume | Notes |
|------------|-------------|-------|
| B2C broad | 1,000+ | High competition |
| B2C niche | 100-1,000 | Moderate competition |
| B2B | 50-500 | Lower volume, higher value |
| Local | 10-100 | Location-specific |
| Long-tail | 10-50 | Low competition, high intent |

### 4. Search Intent Classification

**Intent Types**:

```
Informational (I)
├── "what is cognitive behavioral therapy"
├── "how does anxiety affect sleep"
└── "signs of depression"

Navigational (N)
├── "[brand] login"
├── "[company] contact"
└── "[product] support"

Commercial Investigation (C)
├── "best therapy for anxiety"
├── "CBT vs DBT comparison"
└── "[service] reviews"

Transactional (T)
├── "book therapy appointment"
├── "buy [product]"
└── "psychologist near me"
```

**Intent Mapping Template**:

| Keyword | Volume | KD | Intent | Content Type |
|---------|--------|-----|--------|--------------|
| [keyword 1] | X | Y | I/N/C/T | Blog/Page/Product |
| [keyword 2] | X | Y | I/N/C/T | Blog/Page/Product |

## Keyword Research by Search Engine

### Google Keyword Research

**Process**:
1. Start with Google Keyword Planner for volume
2. Analyze SERP for intent and competition
3. Check Google Trends for seasonality
4. Use Search Console for existing opportunities
5. Analyze PAA for content gaps

### Bing Keyword Research

**Differences from Google**:
- Often less competitive
- Older demographic skews
- More exact match importance
- Voice search via Cortana

**Tools**: Microsoft Advertising Intelligence, Bing Webmaster Tools

### Baidu Keyword Research (China)

**Key Considerations**:
- Simplified Chinese characters
- Local hosting required
- ICP license needed
- Different search behavior

**Tools**:
- Baidu Index (百度指数) - trend data
- Baidu Keyword Planner - volume data
- 5118.com - third-party tool

### Yandex Keyword Research (Russia)

**Key Considerations**:
- Russian language optimization
- Cyrillic character handling
- Regional targeting important
- .ru domain preference

**Tools**:
- Yandex.Wordstat - official keyword tool
- Yandex.Metrica - analytics and keywords

### Naver Keyword Research (Korea)

**Key Considerations**:
- Korean language (Hangul)
- Blog content heavily weighted
- Naver Cafe communities important
- Different content formats

**Tools**:
- Naver Search Advisor
- Naver Keyword Tool
- Naver Data Lab

## Keyword Clustering & Content Mapping

### Topic Cluster Model

```
                    ┌─────────────────┐
                    │   Pillar Page   │
                    │  (Main Topic)   │
                    └────────┬────────┘
                             │
        ┌────────────────────┼────────────────────┐
        │                    │                    │
   ┌────▼────┐         ┌────▼────┐         ┌────▼────┐
   │ Cluster │         │ Cluster │         │ Cluster │
   │ Content │         │ Content │         │ Content │
   │   #1    │         │   #2    │         │   #3    │
   └─────────┘         └─────────┘         └─────────┘
```

### Clustering Process

1. **Group by intent**: Same intent keywords together
2. **Group by topic**: Semantically related keywords
3. **Identify pillar**: Highest volume, broadest topic
4. **Map clusters**: Supporting content for each subtopic

### Content Mapping Template

| Cluster | Pillar Keyword | Supporting Keywords | Content Type | Priority |
|---------|----------------|---------------------|--------------|----------|
| [Topic 1] | [main kw] | [kw1, kw2, kw3] | Pillar page | High |
| [Topic 1a] | [cluster kw] | [kw4, kw5] | Blog post | Medium |
| [Topic 1b] | [cluster kw] | [kw6, kw7] | Blog post | Medium |

## Competitor Keyword Analysis

### Competitor Identification

**Types of Competitors**:
1. **Direct**: Same products/services
2. **Indirect**: Different solution, same problem
3. **SERP**: Ranking for your target keywords

### Analysis Framework

| Analysis | What to Find | Tools |
|----------|--------------|-------|
| Keyword gap | Keywords they rank for, you don't | Ahrefs, SEMrush |
| Content gap | Topics they cover, you don't | Manual + tools |
| Ranking comparison | Position differences | Rank trackers |
| Backlink keywords | Keywords in anchor text | Ahrefs, Moz |

## Long-Tail Keyword Strategy

### Benefits of Long-Tail

| Aspect | Head Terms | Long-Tail |
|--------|------------|-----------|
| Volume | High | Low |
| Competition | High | Low |
| Conversion | Lower | Higher |
| Intent clarity | Vague | Specific |

### Finding Long-Tail Keywords

**Sources**:
- Google Autocomplete (type and wait)
- "People Also Ask" boxes
- Related searches (bottom of SERP)
- Forum questions (Reddit, Quora)
- Customer support tickets
- Review text mining

### Long-Tail Patterns

```
[base keyword] + [modifier]

Modifiers:
- Location: "in Sydney", "near me", "Australia"
- Intent: "how to", "best way to", "tips for"
- Qualifier: "for beginners", "for adults", "for free"
- Comparison: "vs", "or", "compared to"
- Year: "2025", "latest", "updated"
- Format: "template", "checklist", "guide", "examples"
```

## Keyword Prioritization Matrix

### Scoring Framework

| Factor | Weight | Score (1-5) |
|--------|--------|-------------|
| Search volume | 25% | Based on volume |
| Competition/KD | 25% | Inverse of difficulty |
| Business relevance | 30% | Alignment with offerings |
| Conversion potential | 20% | Intent signals |

## Keyword Research Checklist

### Initial Research
- [ ] Define business goals and target audience
- [ ] Generate seed keywords (50-100)
- [ ] Expand using tools and methods
- [ ] Gather metrics (volume, KD, CPC)
- [ ] Classify search intent

### Analysis
- [ ] Analyze competitor keywords
- [ ] Identify keyword gaps
- [ ] Find long-tail opportunities
- [ ] Check SERP features

### Organization
- [ ] Cluster keywords by topic
- [ ] Map to content types
- [ ] Prioritize using matrix
- [ ] Create content calendar

### Ongoing
- [ ] Track rankings weekly
- [ ] Review Search Console monthly
- [ ] Update research quarterly
- [ ] Adjust strategy based on results
